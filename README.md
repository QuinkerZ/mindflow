# 🧠 MindFlow - EEG Monitoring System

## 📌 Overview
MindFlow is a real-time EEG monitoring system that captures brain signals using Arduino Uno and visualizes them on a web dashboard.

## ⚙️ Features
- Real-time brain signal acquisition
- Data processing and filtering
- Web-based visualization dashboard
- Scalable backend (Node.js)

## 🛠️ Tech Stack
- Hardware: Arduino Uno / BioAmp EXG Pill
- Backend: Node.js
- Frontend: HTML, CSS, JavaScript
- Database: (add if using Supabase)

## 🔌 System Architecture
Electrodes → Amplifier → Arduino → Backend → Web Dashboard

## 🚀 Future Improvements
- Live streaming (WebSockets)
- AI-based brain signal analysis
- Mobile app integration
